# Mercado Easy

`CURSO` Sistemas de Informação

`DISCIPLINA` Projeto - Aplicações Web

`SEMESTRE` Primeiro Semestre

  Plataforma digital que terá como intuito a análise dos menores preços dos produtos dos supermercados da região do usuário. O aplicativo terá como função principal um filtro que levará em conta fatores como: os preços dos produtos pesquisados, como também a entrega oferecida pelo estabelecimento. O nosso projeto visa conter uma interface intuitiva, não sendo focada somente em um público-alvo específico, ser também uma tecnologia que auxiliará tanto os mais jovens como o público de maior idade.
